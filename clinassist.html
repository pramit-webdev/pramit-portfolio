ClinAssist AI

ClinAssist is an AI-powered clinical documentation and coding engine that converts doctor notes into structured, ICD-10-coded, FHIR-compatible medical records. 

Doctor note
→ LLM fact extraction
→ ICD-10 semantic engine
→ SOAP + insurance text
→ Database
→ FHIR Bundle

Why it matters

Hospitals, insurers, and ABDM require structured records. Doctors write free-text. ClinAssist bridges that gap.

What I built

  • • FAISS + SentenceTransformer ICD engine
  • • Patient-level visit tracking
  • • FHIR R4 bundle generator
  • • Insurance-ready SOAP output
